SCM Drawing Setup

IntuiCAD SCM would like to take drawing setup back to a simpler time...when one walked over to a plan file and removed a client vellum or mylar and got to work. Simply select the environment and enter the project number.
 

Note that the example above performs an initial layout/model drawing setup AND sets up a revision sketch (named FRED) in less than forty seconds. More SCM videos here...

 

More Features

  1. Setup allows for project borders to be externally referenced for maximum efficiency. Office borders are single-source inserted.
  2. Automates revision sketch tab creation.
  3. Allows user to choose layers, text and dimension styles or none.
  4. Can be customized to manage model space drawing area.
  5. Automatically inserts border attributes.
  6. Sample borders in frequently used sizes with automated diesel strings and dynamic features.
  7. “Custom” border design included as a template for special conditions.
  8. Borders can be customized to your company’s project directory tree.
